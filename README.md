## Sobre a loja

A **Agromundo Fauna e Flora** é uma loja completa em Itajubá (MG) que reúne, em um só
endereço, tudo o que o campo, o pet e o jardim precisam. Desde 2007, oferece produtos de
qualidade e um atendimento especializado que conhece de verdade cada necessidade.

## 💻 Sobre este site

Site institucional moderno e responsivo, desenvolvido com foco em **UI/UX**, **performance**
e **SEO local**, para atrair e converter novos clientes.

### Destaques
- 🎨 Design moderno com a identidade visual oficial da marca
- 🖼️ **Carrossel de fotos reais** da loja no topo (imagens otimizadas)
- 📸 **Seção do Instagram** com posts no estilo do feed que levam ao perfil
- 📱 Totalmente responsivo (celular, tablet e desktop)
- 💬 Botão flutuante de **WhatsApp** para contato imediato
- 🗺️ Mapa da loja e botão "Como chegar" integrados ao Google Maps
- ⚡ Site estático, leve e rápido — sem frameworks pesados
- 🔍 Otimizado para o Google (meta tags, Open Graph e dados estruturados de negócio local)
- ♿ Boas práticas de acessibilidade (HTML semântico, contraste e navegação por teclado)

### Tecnologias
`HTML5` · `CSS3` (design system próprio) · `JavaScript` (vanilla)

### Estrutura
```
agromundo/
├── index.html          # Landing page (todo o conteúdo)
├── css/styles.css      # Estilos e design system
├── js/main.js          # Interações (menu, scroll, animações)
├── assets/             # Logo, favicon e imagens
└── .github/workflows/  # Deploy automático no GitHub Pages
```

## 🚀 Rodando localmente

Por ser um site estático, basta um servidor local simples:

```bash
# Python (já vem no Windows/Mac/Linux)
python -m http.server 5500
# abra http://localhost:5500

# ou com Node
npx serve .
```
> No VS Code, a extensão **Live Server** também funciona (botão "Go Live").

## ☁️ Deploy

O deploy é **automático**: cada `push` na branch `main` publica o site no **GitHub Pages**
através do workflow em [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).
